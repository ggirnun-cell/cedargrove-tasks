# Cedar Grove Capital — Document Brand Guide

Use this branding on any document that might be distributed (Word, PDF, slides).

## Assets

## Colors
- Dark green (headings, table header rows, title): `#334E44`
- Copper (accent rules, subheadings, "CAPITAL" wordmark): `#B36629`
- Cream (table/callout shading): `#F4E4D2`

## Layout conventions (from branded reference doc)
- Centered logo at top of page 1
- Title: bold, dark green
- Subtitle line: "Cedar Grove Capital | <doc purpose>" in italic gray, followed by a copper horizontal rule
- H1: dark green bold; H2: copper bold
- Tables: dark green header row with white bold text; alternating cream row shading; light gray borders
- Footer: "Cedar Grove Capital" left, page number right, copper top border
